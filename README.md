## About
DevBlog is a full-stack blogging platform where users can register, create posts, like/unlike posts, and interact with posts from other users — similar to a mini version of Facebook or Instagram.  
The app includes a dark mode, responsive design, and real-time display of all users' posts.

Live Demo: [https://blog-app-yx85.onrender.com](https://blog-app-yx85.onrender.com)

---

## Features
- User authentication (register/login/logout)
- Create, edit, and delete your own posts
- Like/unlike posts
- View posts from all users
- Dark mode toggle
- Fully responsive UI

---

## Tech Stack
- **Backend:** Node.js, Express
- **Database:** MongoDB (Atlas)
- **Frontend:** EJS templating, HTML, CSS
- **Authentication:** Express-session & bcrypt
- **Deployment:** Render
